今天的主角——又是python

爬取对象

某同城交友网站上，有很多漂亮小姐姐的照片和个人信息，今天将她们的信息作为练习对象，爬取到本地。

爬取信息项

通过请求头，可以看到年龄范围（startage，endage）、性别gender（1代表男，2代表女）、城市cityid、身高范围（startheight，endheight）、婚姻状态（1代表未婚）、学历education、薪资salary这些信息。

总结

1.最开始使用python2.x版本，导致input无法输入汉字。（解决办法：使用python3.x后，问题解决）


2.程序还有漏洞！爬取的信息存储在images&girlinfo文件下（如：E:\py项目\findgirls\girlinfo\Alisa168.txt），但是当某用户的昵称中包含“\”字符时，程序会报错终止，因为文件是以用户的昵称命名的（如：Alisa168），当昵称中包含“\”时，必定会导致找不到文件，从而报错。


3.第一次使用requests、os库，感觉爬取网页信息时很方便。
